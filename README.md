## Getting started

### 如何看代码?

找到src/com/kyson 下的__FileIndex__文件，然后看文件与题目之间的一一对应关系
例如
```java
		map.put("1.1.29", EquivalentKey.class.getName());
```
的意思就是，EquivalentKey 对应的是1.1.29 的习题
